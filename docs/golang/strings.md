# Строки (string) в Go

В Go (Golang) строки являются основным типом данных, представляющим последовательность символов. Они неизменяемы, что означает, что после создания строки ее содержимое не может быть изменено. Это ключевое отличие от строк во многих других языках программирования.

### Представление строк

Строки в Go определяются с использованием двойных кавычек (`" "`). Например:

```go
var greeting = "Hello world"
```

Вы также можете использовать сокращенную нотацию:

```go
greeting := "Hello world"
```

### Характеристики строк

- **Неизменяемость**: После создания строки ее нельзя изменить. Любая операция, которая кажется изменяющей строку, на самом деле создает новую строку.
- **Кодировка UTF-8**: Строки Go закодированы в UTF-8, что позволяет представлять широкий диапазон символов из различных языков.

### Строковые литералы

В Go есть два типа строковых литералов:

1. **Интерпретируемые строковые литералы**: Заключены в двойные кавычки (`" "`), поддерживают управляющие последовательности (например, `\n`, `\"`).
2. **Сырые строковые литералы**: Заключены в обратные кавычки (`` ` ``), не поддерживают управляющие последовательности и могут занимать несколько строк.

### Основные операции со строками

Go предоставляет различные функции для манипуляции строками через пакет `strings`. Некоторые часто используемые функции включают:

| Функция       | Описание                                        |
|---------------|--------------------------------------------------|
| `Compare()`   | Сравнивает две строки лексикографически.        |
| `Contains()`  | Проверяет, существует ли подстрока в строке.   |
| `Split()`     | Разделяет строку на подстроки по разделителю.   |
| `Join()`      | Объединяет элементы среза в одну строку.        |
| `ToLower()`   | Преобразует все символы в строке в строчные.    |
| `ToUpper()`   | Преобразует все символы в строке в заглавные.   |

### Пример кода

Вот пример, демонстрирующий некоторые основные операции со строками:

```go
package main

import (
    "fmt"
    "strings"
)

func main() {
    // Определить строку
    message := "Привет, Golang!"

    // Проверить, содержит ли строка "Golang"
    fmt.Println(strings.Contains(message, "Golang")) // Вывод: true

    // Разделить строку на слова
    words := strings.Split(message, " ")
    fmt.Println(words) // Вывод: [Привет, Golang!]

    // Объединить слова обратно в одну строку
    joined := strings.Join(words, "-")
    fmt.Println(joined) // Вывод: Привет,-Golang!
}
```

### Доступ к символам и рунам

В Go вы можете получить доступ к отдельным байтам строки с помощью индексации, но имейте в виду, что это возвращает байты, а не руны (символы). Чтобы пройти по рунам (что правильно учитывает многобайтовые символы), используйте цикл `for range`:

```go
for index, runeValue := range message {
    fmt.Printf("Символ %c начинается с байта %d\n", runeValue, index)
}
```

### Заключение

Понимание строк в Go имеет решающее значение для эффективного программирования на этом языке. Их неизменяемость и кодировка UTF-8 предоставляют как преимущества, так и вызовы при работе с текстовыми данными. Пакет `strings` предлагает надежную функциональность для эффективной манипуляции и обработки строк.
